AGGIE REUSE STORE AT UC DAVIS

Rebranding an Existing Resource

The Aggie Reuse Store is a second-hand store located on UC Davis' campus. The store uses uses a mutual aid model, which provides all items to be zero-cost for students and community. Their mission is to promote sustainable fashion and creative upcycling by providing a source for basic needs to the overall community while discouraging fast-fashion.

Before this project, I was nervous about using Illustrator and Photoshop since it was my first time. I am glad I challenged myself to start with Adobe software and eventually familiarize myself with the different tools.
